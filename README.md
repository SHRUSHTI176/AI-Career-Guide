# AI_Career_Guide (Phase 1: Text Assistant)

This repo contains a minimal text assistant using Gemini (or simulated mode) + MongoDB and a simple React frontend.

## Quick start
1. Backend:
   - cd backend-ai
   - copy .env.example -> .env and fill MONGO_URI
   - python -m venv venv; source venv/bin/activate
   - pip install -r requirements.txt
   - uvicorn app:app --reload --port 8000

2. Frontend:
   - cd frontend
   - npm install
   - npm run dev

Open frontend URL and test chat.
